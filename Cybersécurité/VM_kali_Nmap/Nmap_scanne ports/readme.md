# Nmap Scanne des ports ouvert sur une machine

nmap               |  Permet d'analyser les ports et de faire un mapping, Exemple : nmap -p 80 192.168.1.40 ici on scanne le port 80 de l'adresse ip , 80/tcp closed http

* VM2 : Metsploitable
