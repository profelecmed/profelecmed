### Les bases du JSON.

JSON (JavaScript Object Notation) est un format d'échange de données léger, facile à lire et à écrire pour les humains, et simple à analyser et générer pour les machines. Il est largement utilisé pour transmettre des données entre un serveur et une application web.
Le format JSON permettant d’enregistrer des données en un unique fichier, à la fois lisible par l’ordinateur et par un être humain.
